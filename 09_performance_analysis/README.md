### Анализ производительности программы

Компиляция:
```bash
nvcc kernel.cu -o kernel
```

Запуск анализа из курса:
```bash
python3 get_nsight.py
```

Скрипт `get_nsight.py` запускает CUDA-приложение с разными значениями `block_size` - от 32 до 1024 с шагом 32. Каждый запуск осуществляется через ncu с сохранением метрик в файл `output.csv`. После завершения запусков скрипт создает 3 файла с графиками: `graph_1.png`, `graph_2.png` и `graph_3.png`

Запуск кастомоного скрипта для анализа:
```bash
python3 my_get_nsight.py --kernel ./kernel
```
